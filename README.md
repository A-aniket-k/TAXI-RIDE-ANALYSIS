# 🚖 Taxi Ride Analytics Dashboard (Light & Dark Theme Editions)

An industry-grade Power BI Data Analytics project focused on tracking, visualizing, and analyzing taxi operational performance, ride completions, and cancellation dynamics. This repository features two completely separate, custom-designed `.pbix` reporting files—one fully engineered for **Light Theme** and another built for **Dark Theme**—to deliver executive-ready stakeholder insights across different visual formats.

## 📊 Business Metrics & Objectives
* **Total Bookings & Operational Volume:** Monitoring overall ride frequency and demand patterns over time.
* **Granular Ride Status Analysis:** Breakdown of successful rides vs. dropped or incomplete trips.
* **Cancellation Diagnostics:** Deep-dive into specific reasons for cancellations from both the customer and driver perspectives.
* **Executive Performance Tracking:** High-level KPIs and multi-dimensional matrices to help business owners optimize fleet allocation and minimize churn.

## 💡 Technical Implementation & Highlights
* **Advanced Data Modeling:** Structuring and cleaning complex transactional taxi data for optimal analytical efficiency.
* **Custom DAX Measures:** Developed targeted calculations for live booking rates, completion percentages, and multi-layered operational performance metrics.
* **Industry-Standard Visual Selection:** Stepped away from basic out-of-the-box configurations to master how specific charts (trend lines, performance KPIs, and custom matrices) are actually applied in real-world corporate analytics.
* **Dual-File Theme Engineering:** Built two distinct standalone dashboards to ensure perfect contrast, visibility, and professional color mapping across both native Light and Dark theme layouts.

## 🛠️ Technologies Used
* **Business Intelligence:** Microsoft Power BI Desktop (Two separate `.pbix` project configurations)
* **Language:** DAX (Data Analysis Expressions)

## 🚀 How to Explore
1. Clone this repository to your local system.
2. Open the **`Taxi_Analytics_Light_Mode.pbix`** file for the crisp, high-contrast corporate layout.
3. Open the **`Taxi_Analytics_Dark_Mode.pbix`** file to explore the sleek, modern low-light layout designed to reduce eye strain.
4. 💡 **Theme Customization:** The provided Power BI files serve as production-ready templates. If your organization requires a different style, you can easily change or customize the underlying themes and visual parameters inside Power BI Desktop to adapt the dashboard according to your specific needs.
5. Review the structural queries and data model inside Power BI Desktop to see the dynamic calculations.
